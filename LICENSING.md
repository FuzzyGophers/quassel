# Quassel IRC Licensing

Quassel has adapted the [REUSE](https://reuse.software/) Software Specification to ensure compliance across
our codebase.

## Installation

### Linux
Use your favorite package manager to install REUSE
```bash
   reuse lint
```

### MacOS (Homebrew)
```bash
   brew install reuse
   reuse lint
```
